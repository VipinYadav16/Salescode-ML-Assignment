Handcrafted Classical Features
  • 1. FFT Spectrum: Analyzes high-frequency spikes and peaks typical of screen pixels and Moiré patterns.
  • 2. LBP Histograms: Captures local micro-texture variations (Local Binary Patterns).
  • 3. Sharpness (Laplacian): Measures defocusing, lens blur, and edge attenuation differences.
  • 4. Noise Residuals: Estimates sensor pattern noise which gets lost/duplicated during recapture.
Latency Target

Latency: ~650 ms on local CPU.
Optimized via center-cropping to (256, 256) and using highly optimized scikit-learn standard linear classifiers.

Cost at Scale

On-Device: $0.00 / free (once compiled to webassembly or native mobile library).
Cloud API: ~$0.05 per 1,000 requests using basic AWS Lambda/serverless CPU hosting.

Goal Accuracy

Target: 95%+ accuracy. Current baseline: 68.3%.
Can be significantly boosted by incorporating advanced frequency/Moiré filtering or lightweight deep networks (MobileNet/SqueezeNet).
